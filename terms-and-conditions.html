How we use cookies

A cookie is a small file which asks permission to be placed on your computer's hard drive. The file is added and the cookie helps analyse web traffic or lets you know when you visit a particular site. Cookies allow web applications to respond to you as an individual. The web application can tailor its operations to your needs, likes and dislikes by gathering and remembering information about your preferences.

We use traffic log cookies to identify which pages are being used. This helps us analyse data about web page traffic and improve our website in order to tailor it to customer needs. We only use this information for statistical analysis purposes and for enhancing your experience whilst using the Store.

Overall, cookies help us provide you with a better website, by enabling us to monitor which pages you find useful and which you do not. A cookie in no way gives us access to your computer or any information about you, other than the data you choose to share with us. You can choose to accept or decline cookies. Most web browsers automatically accept cookies, but you can usually modify your browser setting to decline cookies if you prefer. This may prevent you from taking full advantage of the website.

Extension Privacy Policy

Absolute Commerce extensions contain notification and licencing code to allow us to send messages regarding new releases, updates to modules and other relevant information direct to the Magento admin panel of the installation the module is part of. These notifications use the standard Magento admin messaging mechanism.

Messages are retrieved from the Absolute Commerce server via an external cURL request from the admin server of the installation the module is part of. To opt-out of retrieval of notifications entirely please install the latest version of all Absolute Commerce modules and disable the notification feature in the relevant section of each modules system configuration.

When available the feed of notifications can be accessed directly here: https://absolutecommerce.co.uk/absolute/update

For licencing purposes, the cURL requests to receive notifications are logged on the Absolute Commerce server.

What we collect:

Information What is logged
Host Domain The Magento base URL of the server the Magento admin panel is served from *.
Magento Version The version of Magento the admin panel is running.
Module Name The name of the Absolute Commerce module that made the request.

* The full referrer url of the request is also captured in our webserver access log and is retained for a period of time before being automatically deleted.

For any issues regarding your personal information please contact us.
